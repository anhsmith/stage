# Computes the normalised plateau–Gaussian density used for y = 0

Computes the normalised plateau–Gaussian density used for y = 0

## Usage

``` r
plateau_gaussian_density_left(x, mu, sigma, L)
```

## Arguments

- x:

  vector of x-values

- mu:

  mean location of the Gaussian tail

- sigma:

  transition spread

- L:

  lower truncation
