# Computes the normalised plateau–Gaussian density used for y = 1

Computes the normalised plateau–Gaussian density used for y = 1

## Usage

``` r
plateau_gaussian_density_right(x, mu, sigma, U)
```

## Arguments

- x:

  vector of x-values

- mu:

  mean of the Gaussian tail

- sigma:

  transition spread

- U:

  upper truncation
